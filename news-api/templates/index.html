Lithuanian News API

Welcome to the Lithuanian News API. Use this API to query news article details from Lithuanian news sites like:

  • https://vz.lt
  • https://delif.lt
  • https://15min.lt

Endpoints

  • /articles - Retrieve a list of all news articles

Query Parameters

  • title - filter articles by substring of the title
  • page - filter articles by a news page

Examples

Retrieve a list of all articles:

GET /articles

Retrieve a list of articles with title containing 'Lietuva':

GET /articles?title=Lietuva

Retrieve articles with title containing 'Lietuva' from https://vz.lt:

GET /articles?title=Lithuania&page=vz
